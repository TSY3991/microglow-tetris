Tiny Stairs

小朋友下樓梯

準備開始 左右移動，找到下一層樓梯往下走
分數 0
最高分 0

操作

桌機：方向鍵或 A / D 左右移動，向下鍵或 S 可以直接穿過目前樓梯，空白鍵暫停。

手機：使用下方左右與下樓按鈕，也可以按住畫面左右兩側移動。

遊玩方式

一路往下，不要被畫面頂到

左右移動 控制小朋友踩到下一層平台，平台會慢慢往上移動。
下樓按鈕 按住向下鍵、S 或手機「下樓」可以穿過目前平台，快速掉到下一層。
安全目標 不要被平台推到畫面最上方，也不要掉出畫面底部。
加分方式 每往下一層會加分，吃到星星可以再加分。

遊戲速度會隨分數慢慢提升，適合反覆挑戰最高分。
