Microglow Snake

微光貪吃蛇

準備開始 開始後按住棋盤才會滑行，或用方向鍵 / WASD 控制方向
分數 0
最高分 0
長度 3
遊玩 0

操作

桌機：滑鼠按住棋盤導向，或用方向鍵 / WASD 控制方向，空白鍵暫停。

手機：按住或拖曳棋盤，蛇會平滑朝手指方向滑行。

遊玩方式

按住棋盤控制方向

桌機操作 滑鼠按住棋盤導向，或用方向鍵 / WASD 控制方向，空白鍵暫停，R 重新開始。
手機操作 開始後按住或拖曳棋盤，蛇才會平滑朝手指方向滑行；下方方向鍵也可以輔助操作。
邊界穿越 碰到場地邊界不會結束，會從另一側滑出來。
分數目標 吃到發光能量點會加分、加長，撞到自己的身體才會結束。

遊戲結束後可按「再玩一次」重新開始。
