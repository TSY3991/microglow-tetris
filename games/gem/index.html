Relax Match-3

微光寶石

準備開始 交換相鄰寶石形成 3 連，達成目標分數進入下一關。
分數 0
最高分 0
關卡 1
遊玩 0
關卡目標 0 / 5000

第 1 關目標 5000 分，無時限、無步數限制。

操作

桌機：滑鼠拖曳寶石到相鄰格交換，或點兩格依序選取；箭頭鍵可移動選取游標、空白鍵確定。

手機：點選寶石後再點相鄰寶石，或直接拖曳。

遊玩方式

舒壓寶石對對碰

交換相鄰 將相鄰兩顆寶石交換，形成至少 3 顆同色相連即消除。
4 連 / 5 連 4 連產生「直線消除」道具、5 連 / L / T 形產生「同色清除」道具。觸發時威力加倍。
連鎖加成 消除後寶石下落補新格，若再形成連線會持續連鎖，每階段倍率 ×1.2。
關卡推進 達成目標分數即進入下一關，第 5 關後進入無限模式（每關 +5000）。
無壓力 沒有時間限制、沒有步數限制，無解時自動洗牌。隨時可按提示找一組解。

微光霓虹風，純舒壓向設計。
